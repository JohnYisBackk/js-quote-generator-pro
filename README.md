# Quote Generator PRO ✨

## 🌐 Live Demo

👉 https://your-live-demo-link.com

---

A modern async quote generator built with pure HTML, CSS, and vanilla JavaScript.

This project fetches random quotes from an API and displays them with smooth UI feedback, loading states, and copy-to-clipboard support.

---

## 🚀 Features

- 🔄 Random quote generation
- 🌐 Fetch API integration
- ⚡ Async / await logic
- ⏳ Loading state
- ❌ Error handling
- ✨ Smooth fade animation
- 📋 Copy quote to clipboard
- 🎯 Button disabled state during loading
- 🎨 Modern responsive UI

---

## 🎮 How It Works

1. User clicks **Generate Quote**
2. App fetches a random quote from an API
3. Loading state is displayed
4. Quote text and author update dynamically
5. User can copy the quote to clipboard
